# Структура проекта "Генератор резюме"

## 📁 Общая структура

```
resume-generator/
├── 📄 pom.xml                           # Maven конфигурация
├── 📄 README.md                         # Основная документация
├── 📄 STRUCTURE.md                      # Этот файл - описание структуры
├── 🚀 run.sh                           # Скрипт быстрого запуска
├── 🔨 build.sh                         # Скрипт сборки JAR
├── 📂 src/                             # Исходный код
│   └── 📂 main/
│       ├── 📂 java/                    # Java классы
│       │   └── 📂 com/resumegenerator/
│       │       ├── 📄 Main.java        # Точка входа
│       │       ├── 📂 model/           # Модели данных
│       │       ├── 📂 service/         # Бизнес-логика
│       │       └── 📂 ui/              # Пользовательский интерфейс
│       └── 📂 resources/               # Ресурсы (пустая)
├── 📂 target/                          # Скомпилированные файлы
├── 📂 src_old/                         # Старая простая структура
└── 📄 compile_old.sh                   # Старый скрипт компиляции
```

## 🏗️ Архитектура пакетов

### `com.resumegenerator` (корневой пакет)
- **Main.java** - точка входа в приложение, инициализация UI

### `com.resumegenerator.model` (модели данных)
- **ResumeData.java** - класс для хранения всех данных резюме
  - Поля: имя, email, телефон, адрес, опыт, образование, навыки, тема
  - Геттеры и сеттеры
  - Конструктор с примерами данных

### `com.resumegenerator.service` (бизнес-логика)
- **HtmlGenerator.java** - сервис генерации HTML
  - Статические методы для генерации HTML
  - Поддержка двух тем: профессиональная и креативная
  - CSS стили встроены в HTML

### `com.resumegenerator.ui` (пользовательский интерфейс)
- **LoginFrame.java** - окно авторизации
  - Проверка логина/пароля (newton/newton)
  - Красивый дизайн с подсказками
  
- **ResumeGeneratorFrame.java** - главный редактор
  - Форма редактирования всех полей резюме
  - Выбор темы оформления
  - Кнопка "Отправить" для генерации
  
- **ResumePreviewFrame.java** - предварительный просмотр
  - Отображение HTML резюме
  - Сохранение в файл
  - Автооткрытие в браузере

## 🔧 Файлы конфигурации

### `pom.xml` (Maven)
```xml
<groupId>com.resumegenerator</groupId>
<artifactId>resume-generator</artifactId>
<version>1.0.0</version>
```

## 🚀 Скрипты запуска

### `run.sh` - быстрый запуск
- Автоопределение Maven
- Fallback на ручную компиляцию
- Удобный вывод с эмодзи

### `build.sh` - сборка JAR
- Полная сборка проекта
- Создание исполняемого JAR
- Инструкции по запуску

- `javax.swing.*` - GUI компоненты

## 🎯 Принципы архитектуры

### **Разделение ответственности:**
- **Model** - только данные, без логики
- **Service** - только бизнес-логика
- **UI** - только интерфейс
- **Main** - только инициализация

### **SOLID принципы:**
- Single Responsibility - каждый класс имеет одну задачу
- Open/Closed - легко расширить, не изменяя существующий код
- Dependency Inversion - UI зависит от абстракций, не от деталей

### **Clean Code:**
- Понятные имена классов и методов
- JavaDoc комментарии
- Константы для настроек
- Обработка исключений

## 🔄 Поток выполнения

1. **Main.java** → запуск приложения
2. **LoginFrame** → авторизация пользователя
3. **ResumeGeneratorFrame** → редактирование данных
4. **HtmlGenerator** → генерация HTML из данных
5. **ResumePreviewFrame** → просмотр и сохранение
