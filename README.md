# Генератор резюме

Приложение "Генератор резюме" на языке Java позволяет легко создавать профессиональные резюме в формате HTML с возможностью выбора между двумя стильными темами оформления.

## Особенности

- 🔐 **Безопасный вход в систему** с проверкой учетных данных
- 📝 **Интуитивный редактор** для ввода персональной информации
- 🎨 **Две темы оформления**: Профессиональная и Креативная
- 👁️ **Предварительный просмотр** резюме в HTML формате
- 💾 **Сохранение резюме** в HTML файл
- 🖥️ **Красивый и понятный интерфейс** на Swing

## Как использовать приложение

### 1. Вход в систему
На экране приветствия введите учетные данные:
- **Пользователь:** `newton`
- **Пароль:** `newton`

### 2. Редактирование резюме
После входа вы попадете в редактор резюме, где можете:
- Заполнить личную информацию (имя, email, телефон, адрес)
- Написать краткую информацию о себе
- Добавить опыт работы
- Указать образование
- Перечислить навыки
- Выбрать тему оформления (Профессиональная или Креативная)

### 3. Предварительный просмотр
После заполнения информации нажмите кнопку **"Отправить"** для:
- Сохранения изменений
- Генерации HTML резюме
- Просмотра результата

### 4. Сохранение
В окне предварительного просмотра вы можете:
- Просмотреть сгенерированное резюме
- Сохранить его в HTML файл
- Открыть сохраненный файл в браузере

## Компиляция и запуск

### Требования
- Java 8 или выше
- Любая IDE (IntelliJ IDEA, Eclipse) или командная строка

### Компиляция из командной строки

```bash
# Перейти в папку src
cd src

# Скомпилировать все файлы
javac *.java

# Запустить приложение
java Main
```

### Запуск из IDE
1. Откройте проект в вашей IDE
2. Убедитесь, что папка `src` добавлена как source folder
3. Запустите класс `Main.java`

## Структура проекта

```
src/
├── Main.java                    # Точка входа в приложение
├── LoginFrame.java              # Окно авторизации
├── ResumeGeneratorFrame.java    # Главное окно редактора
├── ResumePreviewFrame.java      # Окно предварительного просмотра
├── ResumeData.java              # Класс данных резюме
└── HtmlGenerator.java           # Генератор HTML
```

## Темы оформления

### Профессиональная тема
- Классический дизайн с синими акцентами
- Четкая структура и читаемость
- Подходит для традиционных профессий

### Креативная тема
- Современный дизайн с градиентами
- Яркие цвета и эмодзи
- Подходит для творческих профессий

## Пример использования

1. Запустите приложение
2. Войдите с логином `newton` и паролем `newton`
3. Заполните поля или отредактируйте предзаполненную информацию
4. Выберите желаемую тему
5. Нажмите "Отправить"
6. Просмотрите результат и сохраните файл

## Автор

Приложение создано как демонстрация возможностей Java Swing для создания десктопных приложений с красивым интерфейсом.
